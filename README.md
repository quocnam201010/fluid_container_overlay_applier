# Fluid Container Overlay Applier
Fluid container overlay applier script. Inspired by Minecraft Hbm's Nuclear Tech Mod fluid container overlay system

## How to use:
If you have a base.png as the empty fluid container and base_overlay.png as the fluid overlay. This is how to generate an image of the base.png with the fluid overlay applied:

 1. Put base.png and base_overlay.png into /input folder
 2. Run the script, insert standard Java hex literal color format, example: 0x888888
 3. The generated image will be generated in /output folder
